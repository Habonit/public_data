# 한국도로공사 고속도로 교통사고 데이터 분석 (1~3일차)

## 개요

한국도로공사 고속도로 교통사고 데이터를 활용하여 탐색적 데이터 분석(EDA)을 수행하고, 설명력 높은 예측 모델을 설계하는 실습 과정입니다.

## 실습 환경

- **환경**: Google Colab
- **언어**: Python

## 데이터 정보

| 항목 | 내용 |
|------|------|
| **데이터 출처** | [공공데이터포털 - 한국도로공사 고속도로 교통사고 데이터](https://www.data.go.kr/data/15145192/fileData.do) |
| **데이터 경로** | `material/1-3/data` |
| **데이터 획득 방법** | 공공데이터포털 계정 생성 후 API Key 발급 → API 호출로 데이터 수집 |

## 커리큘럼

### Day 1: 데이터 수집 및 기초 시각화

데이터를 API로 수집하고 기본적인 시각화와 통계량을 확인합니다.

**학습 내용**
- 공공데이터포털 API를 활용한 데이터 수집
- 데이터 기초 시각화
- 기본 통계량 분석

**실습 노트북**
- [`1_1_한국도로공사_고속도로_교통사고_데이터_불러오기.ipynb`](./1_1_한국도로공사_고속도로_교통사고_데이터_불러오기.ipynb)
- [`1_2_데이터_시각화_및_기본적_통계량.ipynb`](./1_2_데이터_시각화_및_기본적_통계량.ipynb)

---

### Day 2: EDA 및 사고심각도 예측 모델 구축

심화된 EDA를 진행하고, 사고심각도 파생변수를 생성하여 예측 모델을 구현합니다.

**학습 내용**
- 탐색적 데이터 분석(EDA) 심화
- 사고심각도 파생변수 설계
- 분류 모델 구축 및 평가

**실습 노트북**
- [`2_1_EDA_및_시각화_심화.ipynb`](./2_1_EDA_및_시각화_심화.ipynb)
- [`2_2_사고심각도_예측_모델_구축.ipynb`](./2_2_사고심각도_예측_모델_구축.ipynb)

---

### Day 3: 사고발생이정 예측 모델 구축

설명력이 높은 변수를 탐색하고 사고발생이정을 예측하는 모델을 설계합니다.

**학습 내용**
- 주요 변수 탐색 및 선택
- 사고발생이정 예측 모델 설계
- 모델 설명력 분석

**실습 노트북**
- [`3_1_사고발생이정_예측_모델_구축.ipynb`](./3_1_사고발생이정_예측_모델_구축.ipynb)
