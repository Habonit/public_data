# daegu_constitution.md  
대구 공공데이터 기반 Streamlit 시각화 프로젝트 헌법(Constitution)

이 문서는 본 프로젝트의 최상위 규범이며,  
프로젝트의 목적·범위·데이터 구성·사용자 대상·기본 원칙·사용 방법(README)을 정의합니다.  
Spec과 Plan은 본 Constitution의 내용에 반드시 부합해야 합니다.

---

## 1. 프로젝트 개요

본 프로젝트는 대구 지역의 공공데이터(CCTV 정보, 보안등 정보, 어린이 보호구역 정보, 주차장 정보)와  
전국 사고 정보 및 train/test 데이터를 통합적으로 탐색할 수 있는  
**Streamlit 기반 반응형 웹 시각화 애플리케이션**입니다.

각 데이터셋을 개별 탭으로 탐색하고,  
데이터 간의 연관성과 통계적 특징을 확인함으로써  
교육생이 직접 프로젝트 아이디어를 도출할 수 있도록 설계되었습니다.

---

## 2. 프로젝트 목적 및 철학

### 2.1 최상위 목적
- 여러 공공데이터를 한 화면에서 시각화하여 빠르게 이해할 수 있는 환경 제공  
- 교육생이 통계·시각화 기반으로 스스로 문제 정의 및 프로젝트 기획을 수행하도록 돕는 것  
- 복잡한 설치 없이 누구나 Streamlit 앱을 실행할 수 있도록 구성

### 2.2 철학
- 데이터는 "단순 제공"이 아닌 “생각을 촉발하는 도구”여야 한다.  
- Feature Engineering이나 모델링보다, **데이터의 구조·관계·분포를 체감하는 것**을 우선한다.  
- 교육생 스스로 다음 프로젝트 방향을 기획하도록 돕는 시각 정보 제공이 핵심 가치이다.  
- 모든 코드는 단순해야 하며, 파이썬 초보자도 쉽게 이해할 수 있어야 한다.

---

## 3. 프로젝트 범위(Scope)

### 포함 범위
- Streamlit 기반 반응형 웹 화면 구성  
- 각 CSV별 데이터 로딩 및 기초 통계량 출력  
- 그래프 시각화(Plotly 등)  
- 지도 데이터 시각화(Folium, Pydeck 등)  
- 데이터 간 연관성 서술/분석  
- train/test, countrywide_accident, 대구 공공데이터 간 관계 탐색 기능  
- 교육 목적의 가벼운 분석·탐색 수준의 기능

### 제외 범위
- 별도의 백엔드 서버(API) 개발  
- 데이터베이스 구축  
- 머신러닝/딥러닝 모델 학습 기능  
- 대시보드 배포(로컬 실행만 지원)  
- 고급 GIS 처리(좌표 변환, 공간 조인 등 복잡 기능)

## 5. 사용 데이터 명세

다음 데이터 파일은 모두 프로젝트에 포함되며, `/data` 디렉터리에서 로드합니다.

- `대구 CCTV 정보.csv`  
- `대구 보안등 정보.csv`  
- `대구 어린이 보호 구역 정보.csv`

- 위치 기반 지도 시각화  
- 기초 기술통계 계산  
- 데이터 분포/패턴 추정

## 6. 프로젝트 사용 안내 (README 통합)

본 섹션은 프로젝트 사용자가 환경에 상관없이 Streamlit 앱을 실행할 수 있도록 제공하는 실행 설명서입니다.

---

### 6.1 요구 환경
- Python 3.10 이상  
- pip 또는 conda 사용 가능  
- Windows / macOS / Linux 모두 지원

---

### 6.2 설치 방법

1) 프로젝트 디렉터리 준비
```bash
git clone <your_repo_url>
cd project-directory
````

2. 의존성 설치

```bash
pip install -r requirements.txt
```

필요 패키지:

* streamlit
* pandas
* numpy
* plotly
* folium
* geopandas (지도 표현 필요 시)
* pydeck 또는 leafmap 등

---

### 6.3 데이터 파일 준비

```
project-directory/
 ├─ app.py
 ├─ requirements.txt
 └─ data/
      ├─ 대구 CCTV 정보.csv
      ├─ 대구 보안등 정보.csv
      ├─ 대구 어린이 보호 구역 정보.csv
      ├─ 대구 주차장 정보.csv
      ├─ countrywide_accident.csv
      ├─ train.csv
      └─ test.csv
```

* 모든 CSV는 `data/` 폴더 안에 저장
* 파일명/확장자를 변경하면 안 됨

---

### 6.4 실행 방법

```bash
streamlit run app.py
```

브라우저에서 자동 실행되며 기본 주소는:

```
http://localhost:8501
```

---

### 6.5 제공되는 주요 기능

#### 1) 탭 기반 데이터 시각화

각 CSV별 전용 탭에서 다음 기능을 제공:

* 기본 통계량 출력
* 결측치/분포 확인
* 지도 기반 시각화(좌표 제공 시)
* 주요 컬럼 기반 그래프(막대, 산점도 등)

#### 2) 데이터 간 연관성 분석

* train/test와 대구 공공데이터의 위치·분포 비교
* countrywide_accident와 지역 시설 간 패턴 서술
* 프로젝트 발상을 위한 데이터 특징 요약 제공

#### 3) 반응형 UI

* 화면 크기 변화 대응
* 모바일·태블릿에서도 보기 가능

---

### 6.6 자주 발생하는 문제 해결

**Q1. Streamlit 실행 오류**

```bash
pip install --upgrade streamlit
```

**Q2. 지도 시각화 오류**

* 좌표 컬럼명 확인
* folium 또는 geopandas 설치 여부 확인

**Q3. CSV 파일을 찾을 수 없음**

* 반드시 `/data` 안에 넣어야 함
* 파일명에 공백, "copy", 숫자 등이 붙지 않았는지 확인
